Important FAQ

What is the Document Object Model (DOM)?

can be used by scripting language like javascript to manupulate the rendered html in the browser.

using dom, we can add & remove elements, manipulate element's attributes, add events, add animations etc.

How do you select an element from the DOM?

an object called document, which has many properties and methods for accessing and manipulating html elements.

use methods like document.getElementById() or document.querySelector() to select single element and document.querySelectorAll() & many other methods to select multiple elements

What is event delegation in the context of the DOM, and why is it useful?

delegation is an optimization technique for event handling in certain situations.

instead of adding event handlers in each elements, it's better to handle those event in there parent element using a single event. doing this improves the maintainabilty and effeciency of our code

How do you manipulate an element's attributes and styles using the DOM?

after accessing an element, we can use getAttribute(), setAttribute() or removeAttribute() method to manipulates that elements attributes.

we can also use element.style property to style that element. we also have classlist.add() or classlist.remove() method that allows us to add or remove utility classes to style any elements
